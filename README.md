# green_foxes_backend
Backend for /r/webdev weekly project, team green foxes

## Framework
* Dotnet Core 1.0 Web API application

## Build & Run
* Install Dotnet Core 1.0 SDK from https://www.microsoft.com/net/core
* dotnet restore
* dotnet run

## API
Available routes:
* /api/tasks
* /api/tasks/{id}
